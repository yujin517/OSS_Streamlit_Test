# 성향 기반 취미 추천 테스트

## 프로젝트 소개
Streamlit을 활용하여 사용자의 성향에 맞는 취미 유형을 추천해주는 웹 애플리케이션입니다.  
사용자는 20개의 성향 질문에 답하고, 선택 결과에 따라 활동형, 휴식형, 분석형, 감성형, 사교형 중 하나의 유형을 추천받습니다.

## 메인 기능
- 학번 및 이름 첫 화면 표시
- 아이디/비밀번호 기반 로그인 기능
- CSV 파일 기반 퀴즈 데이터 로딩
- `st.cache_data`를 활용한 캐싱 기능
- 최종 성향 유형 및 추천 취미 출력

## 캐싱 설명
퀴즈 문항 데이터는 data/quiz_data.csv 파일에서 불러옵니다.
Streamlit은 사용자의 입력이나 버튼 클릭에 따라 앱이 다시 실행될 수 있으므로, 매번 CSV 파일을 새로 읽지 않도록 @st.cache_data를 적용했습니다.

## 결과 계산 방식
각 선택지는 활동형, 휴식형, 분석형, 감성형, 사교형 중 하나의 유형과 연결되어 있습니다.
사용자가 선택한 답변의 유형 점수를 누적한 뒤, 가장 높은 점수를 받은 유형을 최종 결과로 출력합니다.
